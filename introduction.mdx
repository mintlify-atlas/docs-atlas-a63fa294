## What is prompts.chat?

**prompts.chat** is the world's largest open-source prompt library for AI models. Originally launched in December 2022 as *Awesome ChatGPT Prompts*, it has grown into a full-featured community platform with 154,000+ GitHub stars and thousands of community-tested prompts.

Use prompts.chat to:
- **Browse and discover** high-quality prompts for ChatGPT, Claude, Gemini, Llama, Mistral, and more
- **Create and share** your own prompts with the community
- **Collect and organize** prompts into personal collections
- **Self-host** a private prompt library for your organization with custom branding

## Use prompts.chat your way

<Tabs>
  <Tab title="Web App">
    Browse, create, and manage prompts at [prompts.chat](https://prompts.chat). No installation required — sign in with GitHub or Google to save and share.
  </Tab>
  <Tab title="MCP Server">
    Integrate prompts.chat directly into your AI tools via the Model Context Protocol:
    ```json
    {
      "mcpServers": {
        "prompts.chat": {
          "url": "https://prompts.chat/api/mcp"
        }
      }
    }
    ```
  </Tab>
  <Tab title="CLI">
    Access prompts from the terminal:
    ```bash
    npx prompts.chat
    ```
  </Tab>
  <Tab title="Self-Hosted">
    Deploy your own private instance:
    ```bash
    npx prompts.chat new my-prompt-library
    cd my-prompt-library
    ```
  </Tab>
</Tabs>

## Prompt types

prompts.chat supports multiple prompt formats beyond plain text:

| Type | Description |
|------|-------------|
| **Text** | Standard text prompts for any AI chat model |
| **Image** | Prompts optimized for image generation models (Midjourney, DALL-E, Stable Diffusion) |
| **Structured** | JSON or YAML-formatted prompts for programmatic use |
| **Skill** | Multi-file agent skills for Claude Code and AI coding assistants |
| **Taste** | Personal preference profiles that guide AI responses |
| **Video / Audio** | Prompts for video and audio generation models |
