The [Model Context Protocol](https://modelcontextprotocol.io) (MCP) is an open standard that lets AI tools connect to external data sources and services. By connecting prompts.chat as an MCP server, your AI assistant can search, retrieve, and save prompts directly from your workflow.

**MCP server endpoint:**

```
https://prompts.chat/api/mcp
```

## Connection methods

### Remote (recommended)

Connect directly to the hosted MCP server. No local installation required.

```json
{
  "mcpServers": {
    "prompts.chat": {
      "url": "https://prompts.chat/api/mcp"
    }
  }
}
```

### Local via npx

Run the MCP server locally using the official CLI package.

```json
{
  "mcpServers": {
    "prompts.chat": {
      "command": "npx",
      "args": ["-y", "prompts.chat", "mcp"]
    }
  }
}
```

## Authentication

Read-only MCP tools work without authentication. To use tools that save prompts or skills (`save_prompt`, `save_skill`, etc.), set your API key:

```bash
export PROMPTS_API_KEY=pchat_your_api_key_here
```

Get your API key from [prompts.chat/settings](https://prompts.chat/settings). See [Authentication](/api/authentication) for details.

      "mcpServers": {

          "url": "https://prompts.chat/api/mcp"

    {
      "mcpServers": {
        "prompts.chat": {
